# AI Resume Screening & Candidate Ranking System

An advanced Machine Learning + NLP based web application that analyzes resumes and ranks candidates based on job descriptions.

## Features

- Resume Parsing
- Candidate Ranking
- Job Description Matching
- PDF/DOCX Support
- NLP Processing
- Flask Dashboard

## Technologies

- Python
- Flask
- Scikit-learn
- NLP
- TF-IDF
- Cosine Similarity

## Run Project

```bash
pip install -r requirements.txt
python app.py
